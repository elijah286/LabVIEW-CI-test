Application Launcher.lvlib:Splash Screen.vi

This is the top-level VI of this application. When it runs, it behaves like the splash screen many applications use to keep the user occupied while the application loads into memory. This application is quite small, so there is little overhead, which would make the splash screen just flash for a moment. Because that flash looks funny, there is a three-second delay inserted into this VI so that it stays up for a little bit.

This VI uses a delay-loaded subVI call to load the rest of the application. This trick ensures that the splash screen can load instantly regardless of how large the rest of the VI hierarchy gets, and the loading of the rest of the VIs becomes one step in the load process.
