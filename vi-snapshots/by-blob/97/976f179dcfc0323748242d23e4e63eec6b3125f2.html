Cooler.lvlib:Cooler.lvclass:Get Deadband Offset.vi

The Cooler has a specific desired temperature that it tries to maintain. To prevent the system from switching on and off quickly when the actual temperature is very close to the desired temperature, there is a deadband -- a region of actual temperature values where if the system is on it stays on and if the system is off it stays off. For the Cooler class, this is fixed and not user adjustable. This degree constant is returned by this subVI.
