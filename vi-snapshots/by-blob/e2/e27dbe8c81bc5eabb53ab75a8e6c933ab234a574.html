Level Controller.lvlib:Level Controller.lvclass:Set Output State.vi

This VI specifies the behavior of a level controller when the variable it is monitoring goes above the deadband or below the deadband. It is only invoked in the level controller class when the operation mode switches from one mode to another. The input variable Output State is true if the current value is above the deadband and false if below the deadband.
