Evaporative Cooler.lvlib:Evaporative Cooler.lvclass:Stop Core.vi

In override VIs of Stop Core.vi, an actor should invoke any code needed to stop all the parallel processes that it created during its run. This includes any nested actors that it created and any other asynchronously running VIs, such as a Time-Delayed Message. It may also include all parallel loops in its Actor Core, although it is acceptable to stop those using a local signal on the Actor Core block diagram that is raised after the Call Parent Node finishes.

The <b>final error code</b> input tells the actor what error was set when this actor quit. Zero means there was no error. You can use this to decide whether this should be treated as a Normal Stop or an Emergency Stop for the purposes of signaling other actors. In general, any non-zero value should be treated as an Emergency Stop, though a given actor is free to choose other policies. Note that the particular error code 1608 indicates that this actor was itself stopped by an Emergency Stop and should pass that along.
